## Project Objective
Analyze current job market skill demand using real job listings data.

## Dataset
Source: Manually curated sample dataset based on LinkedIn job postings  
Fields: job_title, company, location, skills

## Analysis Performed
- Skill frequency analysis
- Demand ranking
- Visualization using matplotlib

## Output
- skill_demand.png (bar chart showing skill demand)

## Tools Used
- Python
- Pandas
- Matplotlib
- Git & GitHub
